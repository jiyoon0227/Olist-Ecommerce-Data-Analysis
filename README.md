#  Olist E-Commerce Data Analysis: 물류 병목이 고객 리텐션에 미치는 영향 및 RFM 세분화 전략

## 1.  프로젝트 개요 (Overview)
본 프로젝트는 브라질 최대 이커머스 플랫폼 Olist의 약 11만 건 결제 데이터를 활용하여, 단순한 매출 대시보드 구축을 넘어 **"플랫폼의 성장 정체 원인"**을 데이터로 규명한 비즈니스 분석 프로젝트입니다. 
매출 지표 이면에 숨겨진 물류 인프라의 한계를 진단하고, 이것이 고객의 재구매(Retention)에 미치는 치명적인 영향을 **RFM 고객 세분화 모델**을 통해 정량적으로 증명하여 C-Level 관점의 실질적인 액션 플랜을 도출했습니다.

## 2.  사용 기술 및 도구 (Tech Stack)
* **Database:** MySQL (Workbench)
* **Data Engineering:** 1:N 조인 시 발생하는 Fan-out(데이터 팽창) 리스크 방어, 결측치/이상치 정제, 분석용 Data Mart(View) 구축
* **Analytics:** SQL (Window Functions, CTE, Case When, Date Functions), 파레토 법칙 검증, RFM(Recency, Frequency, Monetary) 모델링

## 3.  핵심 분석 결과 및 스토리라인 (Main Insights)

### 🚨 [Symptom] 1. 외형적 성장의 이면과 정체기 진입
* **블랙 프라이데이의 명암:** 2017년 11월, 전월 대비 **52.37%**의 폭발적인 매출 성장을 기록했으나, 이는 객단가(AOV) 상승이 아닌 단순 주문 건수(Volume) 폭증에 의한 것임을 확인했습니다.
* **정체된 객단가와 성장 둔화:** 2018년 내내 객단가가 130~140 헤알 박스권에 갇혔으며, 2018년 봄을 기점으로 플랫폼 성장이 0%대 및 마이너스로 돌아서는 위기 징후를 포착했습니다.

### 🚚 [Cause] 2. 성장 발목을 잡은 물류 병목 현상
* **통제 불능의 지연율:** 주문이 폭주한 2018년 2~3월, 배송 지연율이 무려 **21.36%**까지 치솟으며 물류 시스템이 마비되었음을 확인했습니다.
* **고객 만족도의 '마지노선':** 배송 소요 기간이 **21일을 초과**하는 순간 평균 리뷰 평점이 4.0에서 3.0점대로 급락하며, 특히 절대적인 속도보다 **'배송 예정일 미준수(+3일 이상 지연)'** 시 평점이 1.85점으로 수직 낙하하는 심리적 임계점을 발견했습니다.
* **핵심 타겟 지역의 붕괴:** 압도적 주문량(1.2만 건)을 발생시키는 리우데자네이루(RJ)의 지연율이 13.47%에 달해, 회사 전체의 CS 평점을 갉아먹는 주원인임을 규명했습니다.

### 💔 [Consequence] 3. 무너진 고객 신뢰와 '밑빠진 독' (RFM 분석)
* **충격적인 97%의 1회성 고객:** RFM 고객 세그먼트 분석 결과, 전체 9.3만 명의 고객 중 2회 이상 재구매한 고객(VIP)은 단 3%에 불과했습니다.
* **가장 뼈아픈 이탈 (High Value One-time):** 평균 객단가 이상의 돈을 쓴 '고가치 잠재 VIP' 그룹이 전체 매출의 **58.49%**를 견인함에도 불구하고, 이들 전원(약 2.2만 명)이 단 1회 구매 후 이탈했음을 데이터로 증명했습니다. 즉, **나쁜 배송 경험이 고가치 고객의 재구매를 원천 차단**하고 있었습니다.

## 4.  데이터 기반 액션 플랜 (Action Plan)

> **"신규 마케팅 예산을 축소하고, 물류 인프라 개선 및 타겟 CRM에 집중 투자해야 합니다."**

1. **Logistics (물류 인프라 재배치):** 절대적인 배송 지연 피해자가 가장 많이 발생하는 리우데자네이루(RJ) 및 지연율 20%를 상회하는 북동부(AL, MA) 지역의 라스트마일(Last-mile) 외주 파트너사를 전면 교체하고 제2 물류 거점을 구축해야 합니다.
2. **Operation (배송 예정일 버퍼 도입):** 고객은 '늦게 오는 것'보다 '약속을 어기는 것'에 더 크게 분노합니다. 지연 위험 지역은 시스템 산출 배송 예정일에 임의로 +3일의 버퍼(A/B 테스트)를 두어 플랫폼 신뢰도를 선제적으로 방어해야 합니다.
3. **CRM (고가치 고객 구출 작전):** `High Value One-time` 세그먼트 고객이 첫 구매 상품을 수령한 직후, 물류 지연에 대한 사과/감사 메시지와 함께 파격적인 '두 번째 구매 전용 VIP 웰컴 쿠폰'을 자동 발송하는 리텐션 자동화 파이프라인을 구축해야 합니다.
